# Web Setup
1. Web Project is developed using Angular
2. From npm install use `npm install --legacy-peer-deps`
3. If you want to install dependency use `npm install --save @progress/kendo-angular-scheduler@latest --legacy-peer-deps`
4. To generate Production artifacts use `ng build --configuration=production`
5. Activate Kendo License using `npx kendo-ui-license activate`
6. Before Activating remove node modules using command `rd /s /q node_modules`
